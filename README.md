# downloading Zlow

To run Zlow, first you will need to download the code for it: 

1. Download git at https://git-scm.com
2. Go to the folder you want to download Zlow to, and open the command line in it by right-clicking, and selecting "Open in Terminal" on windows 11, and on windows 10, holding shift, right-clicking, and selecting "open PowerShell window here". 
3. In the command line, type in ```git clone https://github.com/gioandjake/zlow.git```
